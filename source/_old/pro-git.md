## 简介 ##

第一版出版到现在已经过去了四年。到今天，Git 虽然出现了许多改变，但是还有很多重要的事情一如昨日。因为 Git 核心团队对保持向后兼容性异常固执，所以直到今天大多数核心命令与概念依然有效，但是围绕 Git 的社区还是有一些重大的增加与改变。本书的第二版就是为了更新书籍并讲解那些改动以使其对新用户更有帮助。
